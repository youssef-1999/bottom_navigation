# Mobile Bottom Tab Navigation

This project is a simple mobile bottom navigation bar made for mobile websites and progressive web apps (PWA).

It helps users move between different sections easily using a bottom bar that feels natural and comfortable on mobile.



## What’s included

* Bottom navigation bar fixed at the bottom of the screen
* Text labels for each section
* Clear active state to show the current section
* Smooth switching between sections without page reload
* Mobile-friendly and responsive layout
* Built using plain HTML, CSS, and JavaScript



## Project files


index.html   → Page structure
style.css    → Layout and styling
script.js    → Navigation behavior




## How it works

* Each tab represents a section on the page
* When a tab is clicked:

  * The active tab is highlighted
  * The related section appears
  * Other sections are hidden

Everything happens on the same page with no refresh.



## User experience

* Easy to use with one hand
* Clean and simple design
* Clear navigation between sections
* Suitable for mobile apps and mobile-first websites



## Customization

You can easily:

* Change the tab text
* Add or remove sections
* Adjust colors and spacing
* Replace the demo content with real content


## Ready to use

The navigation is ready to use or integrate into any mobile web project.
